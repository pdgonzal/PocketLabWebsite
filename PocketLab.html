The Project

The vision is to have PocketLab use the phoneâ€™s display to provide instantaneous readings from the on-phone and connected sensors. The user will then be able to monitor the readings of sensors, which will be provided as an onphone graphical output. Such values will be then inputted to a .csv file that can be downloaded for further analysis on a computer. The IOIO-OTG board functions as an input/output device that can be used to control instrumentation and other devices. The application will also include a control module to provide the opportunity to switch relays connected to the interface board (to control temperature, instrumentation or alarms). The board itself is mounted in a small enclosure that provides onboard battery power as well as for an input for standard 5V power supply.
